Quien soy?

Un Desarrollador Back End

Soy tecnólogo en análisis y desarrollo de sistemas de información con conocimiento y experiencia en varias áreas del desarrollo web, testing y atomización de pruebas utilizando patrones de diseño pom y ScreenPlay. Trabajó con varios lenguajes de programación, incluyendo PHP, JavaScript, NodeJS, TypeScript y Java, y desarrolló aplicaciones utilizando frameworks como Spring, NestJS, React, Angular, selenium, serenity DBB. Además, tengo experiencia en el manejo de bases de datos relacionales y no relacionales, como MySQL, SQL Server, PostgreSQL y MongoDB, y en la creación de API para que las aplicaciones front-end puedan interactuar con el servidor. Soy un apasionado por el desarrollo y programación de software, dedicado, responsable y, sobre todo, atento afrontar nuevos retos.
